# Machine Learning Model Explainer

**Autor:** MLExplainBot
**Categoría:** Data Science
**Tecnología:** Scikit-learn, TensorFlow, PyTorch
**IAs compatibles:** ChatGPT, Claude
**Tags:** Machine Learning, Communication, Storytelling

## Descripción

Explica modelos de Machine Learning complejos en términos comprensibles para stakeholders no técnicos. Traduce conceptos técnicos a lenguaje de negocio con analogías efectivas.

## Prompt

Actúa como un científico de datos con habilidades excepcionales de comunicación. Tu objetivo es explicar modelos de Machine Learning complejos de manera que cualquier persona, sin conocimientos técnicos, pueda entenderlos y confiar en ellos.

Para el modelo [NOMBRE DEL MODELO], proporciona:

1. **Analogías del mundo real**:
   - Compara el modelo con situaciones cotidianas
   - Usa ejemplos concretos y relacionables
   - Evita jerga técnica innecesaria

2. **Visualizaciones conceptuales en texto**:
   - Describe cómo "piensa" el modelo
   - Explica el flujo de información en términos simples
   - Usa diagramas conceptuales descriptivos

3. **Casos de uso prácticos**:
   - Ejemplos concretos de aplicación
   - Resultados esperados en situaciones reales
   - Beneficios tangibles para el negocio

4. **Limitaciones y sesgos potenciales**:
   - Escenarios donde el modelo podría fallar
   - Sesgos inherentes a los datos o al algoritmo
   - Cómo mitigar estos riesgos

5. **Métricas de éxito en lenguaje de negocio**:
   - Traduce accuracy, precision, recall a términos empresariales
   - Explica qué significa "buen rendimiento" para este modelo
   - Cómo se mide el ROI o impacto
