**Tecnología:** Power BI, Tableau, Looker
**IAs compatibles:** ChatGPT, Claude
**Tags:** BI, Dashboard, KPIs

## Descripción

Convierte requisitos de negocio en especificaciones técnicas detalladas para dashboards de Business Intelligence. Genera KPIs, fuentes de datos y diseños de visualización.

## Prompt

Actúa como un analista de Business Intelligence senior con experiencia en traducir necesidades de negocio a especificaciones técnicas para dashboards.

A partir de los siguientes requisitos de negocio, genera:

1. **KPIs específicos con fórmulas**: 
   - Nombre del KPI
   - Fórmula de cálculo detallada
   - Lógica de negocio aplicada
   - Umbral de alertas (verde/amarillo/rojo)

2. **Fuentes de datos necesarias**:
   - Tablas/vistas requeridas
   - Campos específicos
   - Joins necesarios
   - Frecuencia de actualización

3. **Diseño de visualizaciones recomendadas**:
   - Tipo de gráfico óptimo para cada métrica
   - Filtros interactivos sugeridos
   - Drill-downs y jerarquías
   - Layout propuesto del dashboard

4. **Consideraciones de granularidad temporal**:
   - Nivel de agregación (diario/semanal/mensual)
   - Períodos de comparación (YoY, MoM, etc.)
   - Dimensiones temporales adicionales
