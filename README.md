# 📊 Data Prompts Hub

**Repositorio comunitario de prompts especializados para profesionales de datos**

Bienvenido a Data Prompts Hub, una colección curada de prompts de alta calidad diseñados específicamente para Data Analysts, Data Engineers, Data Scientists, y profesionales de Business Intelligence.

## 🎯 Características

- ✅ Prompts especializados en el área de datos
- ✅ Organizados por categoría (Analytics, Engineering, Science, BI)
- ✅ Compatibilidad claramente indicada con diferentes IAs
- ✅ Tecnologías y herramientas específicas
- ✅ Fácil de contribuir mediante archivos Markdown
- ✅ Sistema de búsqueda y filtrado intuitivo

## 🚀 Uso

Visita **[https://soyrushian.github.io/data-prompts-hub/](https://soyrushian.github.io/data-prompts-hub/)** para explorar todos los prompts disponibles.

## 🤝 Cómo Contribuir

¡Nos encantaría que agregaras tus prompts! Es muy sencillo:

1. **Fork** este repositorio
2. Crea un nuevo archivo `.md` en la carpeta `prompts/`
3. Usa la siguiente plantilla:
```markdown
# Título del Prompt

**Autor:** TuNombre
**Categoría:** [Data Analytics | Data Engineering | Data Science | Business Intelligence]
**Tecnología:** Herramientas o lenguajes relevantes
**IAs compatibles:** ChatGPT, Claude, Gemini, Copilot (selecciona los que apliquen)
**Tags:** tag1, tag2, tag3

## Descripción

Breve descripción de qué hace el prompt y cuándo usarlo.

## Prompt

[Aquí va el prompt completo, con todas las instrucciones y contexto necesario]
```

4. Agrega el nombre de tu archivo a `prompts/index.json`
5. Crea un **Pull Request**

📖 Lee nuestra [Guía de Contribución](CONTRIBUTING.md) para más detalles.
