# Data Prompts Gallery

Colección curada de prompts de IA para profesionales de datos.

## Contribuir

1. Fork este repositorio
2. Crea un archivo `.md` en la carpeta `prompts/` con este formato:
```markdown
---
title: Título del Prompt
area: analytics|science|engineer|bi
tech: python|sql|spark|powerbi|etc
model: chatgpt|claude|gemini|llama
contributor: tu_usuario_github
---

Contenido completo del prompt aquí...
```

3. Crea un Pull Request

## Estructura del Proyecto
```
data-prompts/
├── index.html
├── styles.css
├── script.js
├── README.md
├── CONTRIBUTING.md
└── prompts/
    └── *.md
```
