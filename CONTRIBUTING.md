# Guía de Contribución

## Cómo agregar un nuevo prompt

1. Crea un archivo `.md` en `prompts/` con nombre descriptivo
2. Usa este formato:
```markdown
---
title: Título del Prompt
area: analytics|science|engineer|bi
tech: python|sql|spark|powerbi
model: chatgpt|claude|gemini|llama
contributor: tu_usuario_github
---

Escribe aquí el contenido completo del prompt...
```

3. Crea un Pull Request

## Categorías

**Áreas:**
- analytics
- science
- engineer
- bi

**Tecnologías:**
- python, sql, spark, powerbi, tableau, etc

**Modelos:**
- chatgpt, claude, gemini, llama
